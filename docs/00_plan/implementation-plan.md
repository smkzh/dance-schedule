## Phase 1: 環境構築

### Step 1: Node.js と Next.js プロジェクトの作成
**学ぶこと**
- Node.js / npm とは何か
- Next.js プロジェクトの構成（どのファイルが何をするか）
- TypeScript, Tailwind CSS の初期設定

**やること**
```
Node.js をインストール
→ npx create-next-app でプロジェクト作成
→ VS Code で開く
→ npm run dev でブラウザに "Hello World" を表示
```

**完了の確認**
- `http://localhost:3000` をブラウザで開いてページが表示される

### Step 10: 時間スロットグリッドを作る
**学ぶこと**
- 配列から UI を生成する方法
- タップで範囲選択する操作の実装（1タップ目=開始、2タップ目=終了→間を一括選択）

**やること**
- カレンダーで日付を選ぶと、09:00〜21:30 の 30 分スロットが縦に並んで展開される
- 1タップ目でスロットの選択開始位置を決め、2タップ目でその間を一括選択できる
- 選択済みのスロットをタップすると選択解除できる

※ドラッグ選択はスマホとPCで別実装が必要なため不採用。タップ範囲選択で代替する。

**完了の確認**
- 日付を選ぶとスロットが展開され、タップで選択できる

### Step 13: 候補日の計算と表示（ナンバー別ビュー）
**学ぶこと**
- 複雑なデータ集計の考え方
- データを加工して UI に表示する方法

**表示の方針**
- 振り付け者が空いている時間帯全体を表示する
- 欠席数はその範囲内で**最も少ない90分ウィンドウ**の人数を表示する
- 欠席者名も表示する

**やること**
- `/numbers/[id]` にナンバーの候補日一覧を表示する
- 振り付け者が空いている日に絞る
- 各日付の「振り付け者が空いている時間帯」と「最小欠席数」を計算して表示する
- 欠席数の少ない順に並べる

**完了の確認**
- 候補日が欠席数の少ない順に表示される
- 各候補に「時間帯・欠席数・欠席者名」が表示される
